# Font Setup - Offline Font Bundle

## Font Dosyalarını İndirme

Font dosyalarını Google Fonts'tan indirmek için:

### Fraunces Font (w600, w700)
1. https://fonts.google.com/specimen/Fraunces adresine gidin
2. "Download family" butonuna tıklayın
3. İndirilen ZIP dosyasını açın
4. Şu dosyaları `assets/fonts/` klasörüne kopyalayın:
   - `Fraunces[wght].ttf` (variable font - önerilen, daha küçük)
   - VEYA `static/ttf/Fraunces-SemiBold.ttf` ve `Fraunces-Bold.ttf`

### Inter Font (w400, w500, w600)
1. https://fonts.google.com/specimen/Inter adresine gidin
2. "Download family" butonuna tıklayın
3. İndirilen ZIP dosyasını açın
4. Şu dosyaları `assets/fonts/` klasörüne kopyalayın:
   - `Inter[wght].ttf` (variable font - önerilen, daha küçük)
   - VEYA `static/ttf/Inter-Regular.ttf`, `Inter-Medium.ttf`, `Inter-SemiBold.ttf`

## Font Boyutları (Tahmini)

**Variable Font (Önerilen - Daha Küçük):**
- Fraunces[wght].ttf: ~150-200 KB
- Inter[wght].ttf: ~150-200 KB
- **Toplam: ~300-400 KB** ✅ (1MB'ın altında)

**Static Fonts (Alternatif):**
- Fraunces-SemiBold.ttf: ~150 KB
- Fraunces-Bold.ttf: ~150 KB
- Inter-Regular.ttf: ~150 KB
- Inter-Medium.ttf: ~150 KB
- Inter-SemiBold.ttf: ~150 KB
- **Toplam: ~750 KB** ✅ (1MB'ın altında)

## Öneri

**Variable font kullanın** - Daha küçük boyut ve tüm weight'leri tek dosyada!
